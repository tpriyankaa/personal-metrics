# Priyanka's Dashboard

A personal productivity dashboard — tasks, reports, and activity tracking.

## Run locally

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually http://localhost:5173).

## Build for deployment

```bash
npm run build
```

This outputs a static site to `dist/`, which you can deploy to Vercel, Netlify, or GitHub Pages.

Data is saved in the browser's localStorage, so it persists between visits on the same device/browser.
